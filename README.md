# annotator

This is frontend SPA for webapiner server.

## How to run it

1. apt-get install nodejs npm git
1. git clone this
1. cd webapi3-ui-annotator
1. npm i

## How to build
 nodejs --max-old-space-size=8192 moonboots-build.js
